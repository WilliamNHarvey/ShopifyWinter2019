Query

Every GraphQL schema has a root type for both queries and mutations. The query type defines GraphQL operations that retrieve data from the server.

Fields

shops ([Shop!]!)

Get shops. Can provide IDs or names to get only shops with those IDs or names.

Argument Type Description
id [Int!]
name [String!]
allShops ([Shop!]!)

Get all shops.

products ([Product!]!)

Get products. Can provide IDs to get only products with those IDs.

Argument Type Description
id [Int!]
allProducts ([Product!]!)

Get all products.

orders ([Order!]!)

Get orders. Can provide IDs to get only orders with those IDs.

Argument Type Description
id [Int!]
allOrders ([Order!]!)

Get all orders.

lineItems ([LineItem!]!)

Get line items. Can provide IDs to get only line items with those IDs.

Argument Type Description
id [Int!]
allLineItems ([LineItem!]!)

Get all line items.
