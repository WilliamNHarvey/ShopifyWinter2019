createProduct

Create a product

Input fields

shop (Int!)
price (Float)
currency (String)
lineItems ([Int!])

Return fields

id (Int)
createdAt (Int)
updatedAt (Int)
price (String)

The price of the product and its line items

currency (String)

The currency of the product's and its line items' price

shop (Shop)

The shop the product belongs to

lineItems ([LineItem!])

Get the line items belonging to the product. Can provide IDs to only get which of those line items belong to the product.

Argument Type Description
id [Int!]
